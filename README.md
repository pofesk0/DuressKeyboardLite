EN: This app is a simplified version of DuressKeyboard app. This app has higher target SDK (because it uses another mechanism for wipe data), but fewer features.

This is keyboard to prepare for emergency situations. When used, it can erase data (reset the phone settings) when you enter a special code (that you set your in advance.). Reset code works only on the lock screen. It help if in the future someone tries to force you to enter a password. In that case, you enter the reset code instead password and press the Enter arrow (⏎).

RU: Это приложение устроенная версия приложения DuressKeyboard. Оно имеет более высокий target sdk (потому что использует другой механизм сброса данных), но меньше функций.

Это клавиатура для подготовки к экстренным ситуациям. Когда используется, может стирать данные (сбрасывать настройки телефона) при вводе специального кода, (который вы устанавливаете заранее). Код сброса работает только на экране блокировки. Это поможет, если кто-то попытается заставить вас ввести пароль. В таком случае вы вводите код сброса вместо пароля и нажимаете стрелку Enter (⏎).
